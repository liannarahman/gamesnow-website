# GamesNow Website

## Overview
GamesNow is a website created for a Human-Computer Interaction course. The project focused on designing an intuitive interface, creating multiple prototypes, and improving usability based on feedback.

## What I Worked On
- Conducted user interviews to identify usability needs
- Used HTML embeds in Wix to customize site layout and functionality
- Improved usability through iterative testing and feedback
